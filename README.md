# ai-pm-learning-log

Public learning log documenting my path to an AI Product Manager role. I learn by building, so this repo is the running record of the artifacts I produce: product teardowns, SQL and analytics exercises, prompt frameworks, evaluation work, and plain-language notes.

## Why this exists
I am building demonstrable AI product skills in public. Each folder holds a concrete artifact, not passive notes.

## How it is organized
teardowns  product and AI-feature teardowns using a fixed five-field template
sql        product-analytics queries, including window functions
prompting  reusable prompt frameworks with before-and-after examples
evals      evaluation sets and results for AI features
notes      plain-language explainers of AI concepts

## Currently working toward
AI Product Manager roles, with a focus on evaluation, agentic systems, and financial-services AI.
